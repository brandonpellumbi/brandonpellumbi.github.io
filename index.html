ForgeDB

Database Engine

A relational database engine built from scratch in C. Implements B+ tree storage with slotted pages and automatic splits, a hand-written recursive descent SQL parser, buffer pool with LRU/clock eviction, write-ahead logging for ACID transactions, and a query executor with WHERE, ORDER BY, LIMIT, DISTINCT, and LIKE.

C B+ Trees SQL Parser Buffer Pool WAL ACID

~6,000 lines of C 14 tests Zero dependencies

ForgeHTTP

HTTP Server

An HTTP/1.1 web server built from scratch using POSIX sockets and pthreads. Features a TCP accept loop, hand-written HTTP parser, thread pool with bounded ring-buffer queue, request router with path parameters and wildcards, static file server with 35+ MIME types, keep-alive connections, and atomic server stats.

C TCP Sockets Thread Pool HTTP/1.1 Request Routing Static Files
Source Code
~3,500 lines of C 18 tests Zero dependencies

ForgeShell

Unix Shell

A Unix shell built from scratch implementing core OS concepts: process forking with fork()/execvp(), multi-stage pipelines using pipe()/dup2(), I/O redirection, job control with fg/bg and signal handling (SIGINT, SIGTSTP, SIGCHLD), quoting with variable expansion, conditional execution, and 14 built-in commands.

C fork/exec Pipes Signals Job Control Process Groups
